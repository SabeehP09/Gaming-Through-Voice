📊 HOW IT WORKS:

ENROLLMENT:
1. Capture 5 face images (front, left, right, up, down)
2. Generate 128-D embedding for each
3. Store in database (2.5KB total)

AUTHENTICATION:
1. Capture face image
2. Generate 128-D embedding
3. Compare with stored embeddings
4. Return best match + confidence

COMPARISON:
• Uses Euclidean distance
• Threshold: 0.6 (iPhone-level)
• Distance < 0.6 = Match
• Distance > 0.6 = No match
