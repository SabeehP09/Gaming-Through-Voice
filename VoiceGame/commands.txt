# Voice Commands for Mr Racer
# Using Vosk speech recognition

# === SYSTEM COMMANDS ===
open mr racer
play mr racer
start mr racer
quit game
close game

# === NAVIGATION (works on multiple screens) ===
back
next

# === MAIN MENU ===
watch
game mode
garage
settings
daily rewards

# === GAME MODE SELECTION ===
challenge
career race
career
endless
free ride
chase

# === SETTINGS ===
close settings
sound
maria voice
auto
manual
km
miles
help
bg music increase
bg music decrease
sound fx increase
sound fx decrease

# === HELP ===
close help

# === GAMEPLAY CONTROLS (keyboard keys) ===
left          -> Left Arrow
right         -> Right Arrow
accelerate    -> Up Arrow
gas           -> Up Arrow
brake         -> Down Arrow
horn          -> H key
camera        -> C key
pause         -> Screen click
straight      -> Release steering
release       -> Release steering
